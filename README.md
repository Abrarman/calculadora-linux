# Projeto Calculadora Linux

## Arquivos

- calculadora.py → código em Python
- calculadora.sh → script executável
- comandos.txt → comandos usados

## Como executar

chmod 744 calculadora.sh

./calculadora.sh

## Descrição

Calculadora que realiza:

- soma
- subtração
- multiplicação
- divisão
- potência
- comparação
- verificação de paridade
